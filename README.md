# nitrofont
NitroFont is a hobbyist, high-quality soundfont initially released on August 14, 2025, on the Polyphone website (https://www.polyphone.io/en). I had a dream of creating a high-quality soundfont since at least 2022, but my soundfont creation skills were so-so back then. However, given that I can actually create good-sounding soundfonts, I was able to make my soundfont go public.

You're free to suggest additions or changes to the soundfont, or you can report an issue here.
